# QOL Lite
QOL Lite acts as a QOL Lock replacement to avoid 1% low issues. It includes the basic mods included in QOL Lock without the big performance hit.

YOU CANNOT HAVE ANY OF THE MODS INSTALLED ALREADY INCLUDED IN QOL Lite OR USE QOL LOCK WHILE USING QOL Lite OR IT WILL CRASH OR JUST NOT WORK!

QOL Lite MUST ALSO BE #1 PRIORITY IN YOUR MOD LOAD ORDER AT ALL TIMES!

## Mods Included in QOL Lite:

    Top Bar Plus

    Better Map

    Advanced Testing Tools In Hideout

    Menu (for queuing while in Custom Servers or Hideout)

    Statlocker

    Stats in Shop

    Clean Damage Indicator

    Vindicta Scope Downscale

    Optimized McGinnis Wall

    Optimized Soul Containers

    Sinner's Light Fix

    4;3 Fix

    4;3 Option in Video Settings
